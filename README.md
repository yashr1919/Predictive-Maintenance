Predictive Maintenance Dashboard
Overview

This project is a real-time Predictive Maintenance Dashboard designed to monitor machine health, predict equipment failures, and estimate Remaining Useful Life (RUL).

The system collects sensor data from ThingSpeak AWS, processes machine condition parameters, and visualizes equipment health through an interactive web dashboard.

Features
Real-time sensor monitoring
Machine health prediction
Remaining Useful Life (RUL) estimation
SVM-based health assessment
Critical and warning alerts
Interactive charts and visualizations
ThingSpeak AWS integration
Responsive dashboard UI
